Overview

You are playing a turn-based hex-grid survival game. Your goal is to accumulate score points. Use three browser tabs simultaneously for efficient play.

Tab Setup (Required)

TabURLPurpose
Tab 1http://192.168.4.70/Game UI — movement, actions
Tab 2http://192.168.4.70/stateLive JSON — score, position, supplies, day
Tab 3http://192.168.4.70/state?pid=0Player map view — visible hexes, terrain, resources

⚠ Open Tab 2 and Tab 3 in separate windows — navigating away from Tab 1 will disconnect your WebSocket session and reset the game.

💡 Preferred method: Read game state without navigating at all — call window.__gameState via browser JavaScript console or automation tools. This gives you full state (position, supplies, visible cells) without leaving the game tab.

Tab 3 only works when the player is connected (Tab 1 is open and a character has been selected). First player is player zero: /state?pid=0. The view object will be absent if the player is not connected — go to Tab 1 and select a character first.

Connecting / Spawning

  1. Navigate Tab 1 to http://192.168.4.70/
  2. The character select screen will appear. Click ▶ SELECT GUIDE (pid:0). Guide has the best NAVIGATE skill (●●) and a trait that reduces movement cost for companions.
  3. After selecting, you spawn with LL:7 (full health), 6 MP, and score 0.
  4. Immediately refresh Tab 3 (/state?pid=0) to read your starting visible cells.

Reading window.__gameState

The fastest way to read state without navigating. Open the browser console and type:

window.__gameState

Returns: myId, day, tc (threat clock), visR, me (all your stats), players (others online), visibleCells (all known map cells with terrain/resource/shelter data).

Reading /state?pid=N

Each cell in the cells array contains:

{ "q": 14, "r": 10,
  "dq": 0, "dr": -3,       // offset from player
  "terrain": 4,
  "terrainName": "Broken Urban",
  "shelter": 0,             // 1 if shelter present
  "resource": 5,
  "resourceName": "scrap",
  "amount": 2,
  "footprints": 0           // 0=NEVER VISITED → exploration bonus }

Turn Structure

  1. Wake up — Click map or dismiss REST popup. Starts with full MP (4–6).
  2. Check supplies — Read /state or window.__gameState.me for food, water, ll, mp, rad, scrap.
  3. Choose ONE action — Only one action slot per day:
    • FORAGE (2 MP) — Scrub/Marsh/Rust Forest → food + 3 pts
    • SCAVENGE (2 MP) — Broken Urban/Glass Fields/Flooded Ruins → scrap + 5 pts
    • COLLECT WATER (1 MP) — Marsh/Flooded District → water + pts
    • BUILD SHELTER (1 MP, 2 scrap) — stops LL exposure, 4–8 pts
    • SURVEY (1 MP) — expands visibility, 2 pts
  4. Move — Use remaining MP toward nearest unvisited high-value hex.
  5. REST — Click REST when MP is 0. Click once, then click map to dismiss dawn popup. Do NOT click REST multiple times — each click advances a full day.

Survival Rules

  • LL drops by 1 each day if no shelter and exposed. LL=0 → death.
  • Shelter: Build when scrap ≥ 2 and LL ≤ 4. Costs 2 scrap, stops daily LL bleed, gives 4–8 pts. Shelters are permanent.
  • Food/Water: Each REST consumes 1 food and 1 water. Thresholds: Food at 4, 2. Water at 5, 3, 1. Each downward cross costs 1 LL. FORAGE every 2–3 days minimum.
  • Radiation: Glass Fields and Ash Dunes give +RAD. RAD≥4 = Rad-Sick. RAD≥7 = Dusk Check risk. Move through quickly.
  • Rust Forest: visR drops to 0 inside — /state?pid=N view returns only current cell. Avoid unless foraging.

Point-Scoring Priority

  1. Move into unvisited Flooded Ruins / Settlement → ~30 pts
  2. Move into unvisited Broken Urban / Glass Fields → ~10–20 pts
  3. SCAVENGE on Broken Urban or Glass Fields → +5 pts + scrap
  4. BUILD SHELTER (Improved) → +8 pts + stops exposure
  5. FORAGE → +3 pts + food
  6. SURVEY → +2 pts + reveals cells in /state?pid=N
  7. COLLECT WATER → minor pts + water

Workflow Loop Summary

EACH TURN:
1. Read window.__gameState (or refresh /state?pid=N in separate window)
2. Filter visibleCells: footprints==0 AND terrain IN [4,5,6,9]
3. Pick nearest unvisited high-value cell
4. Use action slot: SCAVENGE if on SV≥1 terrain, else FORAGE if low food
5. Move toward target using remaining MP
6. If MP=0 → REST once → click map to dismiss dawn popup
7. If LL ≤ 4 and scrap ≥ 2 → BUILD SHELTER next action slot
8. Repeat
